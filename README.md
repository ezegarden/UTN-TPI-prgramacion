# Sistema de Gestion de Paises

Trabajo Practico Integrador - Programacion 1
Tecnicatura Universitaria en Programacion - UTN

## Descripcion

Aplicacion en Python que permite gestionar informacion sobre paises del mundo. Lee datos desde un archivo CSV y ofrece funciones de busqueda, filtrado, ordenamiento y estadisticas.

## Como ejecutar

1. Clonar el repositorio
2. Abrir una terminal en la carpeta del proyecto
3. Ejecutar el comando: python paises.py

## Opciones del menu

1. Agregar pais - agrega un nuevo pais con todos sus datos
2. Actualizar poblacion y superficie - modifica datos de un pais existente
3. Buscar pais por nombre - busqueda parcial o exacta
4. Filtrar paises - filtra por continente, rango de poblacion o superficie
5. Ordenar paises - ordena por nombre, poblacion o superficie
6. Ver estadisticas - muestra promedios, maximos, minimos y totales por continente
7. Salir

## Ejemplo de uso

Al iniciar el programa se cargan automaticamente 20 paises desde el archivo paises.csv. Luego se puede navegar por el menu seleccionando el numero de la opcion deseada.
